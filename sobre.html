A capsula do tempo é um programa com o intuito de, além de estudar o trabalho com api, mostrar os principais acontecimentos em um determinado ano.

Utilizamos Electron, uma framework que integra as linguagens html, css, e javascript para fazer o programa. Electron tem como base o chromiun(O google chrome open source), e hoje é utilizado por várias empresas.
